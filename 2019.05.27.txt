JSP 
<%@  %> : 디렉티브, 페이지 설정
<%   %> : 스크립트 릿, 자바 코드 작성 (지역변수)
<%!  %> : 디클레이션(함수), 선언(전역변수)
<%=  %> : 익스프레션, 표현식

객체지향언어
1. 객체 = 속성 + 기능
객체+객체+객체....=> 클래스(속성+기능)
2. 상속
3. 다형성(오버라이딩)
4. 추상화 
5. 인터페이스 : 다중상속의 형식을 취함
추상화는 일반클래스처럼 상수와 변수를 가지고 있으면서 추상클래스도 있는 반면
인터페이스는 추상화 메소드만 가지고 있음.
